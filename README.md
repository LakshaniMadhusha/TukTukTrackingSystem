# Real-Time Tuk-Tuk Tracking API

This is a RESTful Web API for real-time tuk-tuk tracking and movement logging for Sri Lanka Police.

## Technologies

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Swagger
- AWS Elastic Beanstalk

## Main Features

- User authentication
- Role-based access control
- Province management
- District management
- Police station management
- Tuk-tuk registration
- GPS location update API
- Live/latest location API
- Historical movement log API
- Province, district, and police station filtering
- Simulated data for demonstration
